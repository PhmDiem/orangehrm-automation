# OrangeHRM Automation Testing

Dự án Automation Testing cho OrangeHRM sử dụng Python + Selenium + PyTest + POM.

## Tech Stack
- Python
- Selenium WebDriver
- PyTest
- Allure Report
- Page Object Model (POM)

## Cấu trúc Project
- `pages/` : Chứa các Page Object
- `tests/` : Chứa test cases
- `data/` : Test data (CSV, JSON...)
- `utils/` : Helper functions
- `screenshots/` : Ảnh chụp màn hình khi test fail
- `reports/` : Báo cáo Allure
